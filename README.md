 THE LAST SIGNAL 
 
## Project Overview

This project focuses on solving a sensor placement optimization problem in a 2D grid-based environment. The objective is to maximize target coverage while efficiently using a limited number of sensors under environmental constraints such as obstacles and sensor range limitations.

To solve this optimization problem, local search and evolutionary algorithms were used to explore the search space and identify near-optimal sensor configurations.

## Features

* Optimized sensor placement in a constrained grid world
* Maximized target coverage while minimizing resource usage
* Evaluated algorithm performance across different map configurations
* Compared convergence behavior and search quality

* Genetic Algorithm
